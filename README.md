# Health Link

## 📱 Overview
Health Link is a healthcare management system built using Flutter. The app allows patients to book appointments with doctors from various specializations, manage their health records, and access medical information. Additionally, the app provides features for doctors to manage their appointments and for administrators to oversee the system.

## 🎯 Features
- **Patient Module:**
  - Create and manage patient accounts.
  - Search and book appointments with doctors from different specializations.
  - View appointment history and receive reminders.

- **Doctor Module:**
  - Create and manage doctor profiles.
  - View and manage appointment schedules.
  - Update availability status.

- **Admin Module:**
  - Manage users (patients and doctors).
  - View system analytics and monitor appointments.

## 🛠️ Tech Stack
- **Framework:** Flutter
- **Backend:** Firebase& .NET
- **Database:** Firebase Realtime Database
- **State Management:** Bloc
- **Authentication:** Firebase Auth
- **Deployment Platforms:** Android & iOS

## 🚀 Getting Started
### Prerequisites
Make sure you have the following installed:
- Flutter SDK
- Android Studio or Xcode
- A Firebase project configured with Firestore and Firebase Auth

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Abdalrahman1723/Health-care-management-app.git
   cd health-link
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Configure Firebase using `flutterfire_cli`:
   ```bash
   flutterfire configure
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## 🔑 Firebase Configuration
Ensure that you have the Firebase configuration files (`google-services.json` for Android and `GoogleService-Info.plist` for iOS) in the appropriate directories.

## 📖 Usage
1. Register or log in as a patient or doctor.
2. Navigate through the modules to book appointments, manage schedules, and more.
